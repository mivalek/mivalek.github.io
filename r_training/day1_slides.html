Why are we here?

R

  • Programming language mainly for statistical analysis, data processing and visualisation
  • Started gaining popularity among statisticians, bioinformaticians, psychologists… in the noughties
  • All but replaced proprietary software (SPSS, STATA, MPlus)
  • Open source and free
  • Implements cutting-edge statistical methods

RStudio

  • Integrated Development Environment (IDE) for the R language
  • Not necessary for working in R but very useful!
  • Provides interface with other languages
  • Many useful tools and add-ins
    • Support for R Markdown
    • Version control (e.g., GitHub)
    • R projects

What can you do with R?

  • Load/clean/transform your data
  • Run virtually any statistical analysis
  • Create beautiful, publication-ready plots
  • Write papers
  • Create slides/presentations/lectures
  • Write teaching materials
  • Make websites
  • Blog posts
