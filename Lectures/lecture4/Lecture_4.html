Modelling the world

What are models?

  • Representations of reality
  • Abstractions – They focus on specific aspects and disregard others
  • Generalisations – They often aim to describe not just one instance of a category but the relevant features of the category itself

Statistical models

  • In essence a mathematical model – expresses relationships between variables using equations
  • Makes use of assumptions in order to generalise from sample to population
  • Quantifies the uncertainty related to random sampling from populations
  • There is always a discrepancy between what the model predicts and the data
    • If a model fits the data perfectly, it is a bad model

Models as best guesses

  • You can think of statistical models as best guesses given certain amount of information available
  • If you have a sample of a randomly distributed variable, what is the best guess about the value of a random draw from the same population in the absence of any other information?
      • The mean
  • In other words, if you are my sample and I know each one of your ages, what’s the best guess I can make about the age of a random person from the same population?
  • Why mean? highest probability density!

Modeling relationships

  • Though basic descriptive statistics are models, they’re not very good because they are naïve – they don’t make use of any additional information
  • Often, we want to model relationships between things, e.g. outcome on the one hand and experimental manipulation/independent variable on the other
    • What is the best guess I can make about the value of the outcome, if I know something about the predictor?
  • All the tests that include at least one continuous variable that we will talk about model this kind of a relationship as a line – this is the linear model

Some vocab

  • Prediction is the value of outcome produced by the model
    • prediction = model
    • \(\hat{vocab} = 224 + 168.38\times age - 0.38\times age^2\)
    • for age of 20 years (240 months): \(18286 = 224 + 168.38\times 240 - 0.38\times 240^2\)
  • Observation is the actual observed value in the data
    • observation = model + residual
    • \(vocab = 224 + 168.38\times age - 0.38\times age^2 + \varepsilon\)
  • Residual is then the difference between what the model predicts and what we observe
    • \(\varepsilon = \hat{y} - y\)
  • Is this clear?

Accuracy/generalisability trade-off

  • We want a good fit (small residuals) but not at all costs (too many parameters)
  • The more specific a model is to particular data, the less well it generalises
  • There is a trade-off between how well a model fits the data and how parsimonious it is (i.e., how many parameters it includes)

 Take-home message

  • Statistical models are representation of the world
  • We can model relationships between variables
  • Linear model models an outcome variable as a linear combination of predictor varialbes
  • Even simple models, e.g., mean can be seen as linear models (slope = 0)
  • Models with a perfect fit are useless
  • More complex models will always fit better, this is not always good

“All models are wrong but some are useful” ~ George Box
